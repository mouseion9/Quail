\headerbox{SET-SYSTEM-DEFAULT-MENUBAR}{}{FUNCTION}
 
{\bf Description}

Sets the *system-default-menubar* to be the value of the optional argument menubar ( defaults to the Macintosh's *default-menubar* ) .

 
{\bf Lambda List}

(\&OPTIONAL MENUBAR)

{\bf MENUBAR}
